# Nuevo Horizonte

Centro Educativo Interdisciplinario Terapéutico — Sitio web institucional.

## Descripción

Sitio web estático del centro **Nuevo Horizonte**, ubicado en Rosario, Santa Fe. El sitio presenta los servicios terapéuticos y educativos del centro, su equipo profesional, historia, galería, testimonios y un formulario de contacto.

## Servicios

- Psicología
- Psicopedagogía
- Fonoaudiología
- Terapia Ocupacional
- Musicoterapia
- Psicomotricidad
- Estimulación Temprana
- Apoyo Escolar
- Integración Escolar
- Recreación Terapéutica
- Talleres Grupales
- Orientación Familiar

## Estructura del proyecto

```
├── index.html          # Página principal
├── css/
│   ├── reset.css       # Estilos de reseteo
│   ├── variables.css   # Variables de diseño (colores, tipografía, etc.)
│   ├── global.css      # Estilos globales
│   ├── header.css      # Navegación
│   ├── hero.css        # Hero / portada
│   ├── about.css       # Sección "Sobre Nosotros"
│   ├── services.css    # Sección de servicios
│   ├── professionals.css # Equipo profesional
│   ├── gallery.css     # Galería
│   ├── testimonials.css # Testimonios
│   ├── faq.css         # Preguntas frecuentes
│   ├── contact.css     # Formulario de contacto
│   ├── footer.css      # Pie de página
│   └── responsive.css  # Adaptación móvil
├── js/
│   ├── menu.js         # Menú hamburguesa
│   ├── slider.js       # Carrusel de testimonios
│   ├── faq.js          # Acordeón de preguntas
│   ├── scroll.js       # Animaciones al scroll
│   └── main.js         # Funcionalidades generales
└── assets/
    ├── logo/           # Logo del centro
    └── img/            # Imágenes del equipo
```

## Tecnologías

- HTML5 semántico con Schema.org y Open Graph
- CSS3 con variables, metodología BEM y diseño responsivo
- JavaScript vanilla (sin frameworks)
- Font Awesome para iconografía
- Google Fonts (Poppins + Nunito)

## SEO y accesibilidad

- Meta tags optimizados para buscadores y redes sociales
- Datos estructurados JSON-LD (LocalBusiness)
- Roles ARIA, atributos `aria-label` y navegación por teclado
- Diseño responsivo adaptable a dispositivos móviles
